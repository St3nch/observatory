# The Observatory

The Observatory is a standalone planning project for an evidence-only SEO/GEO/SERP visibility database and LLM-readable evidence layer.

It stores observations, not strategy.

Current status: early planning / doctrine / roadmap shaping.

## What The Observatory Is

The Observatory is the future evidence substrate for SEO / GEO / SERP / visibility intelligence across the owner's projects.

It is intended to store longitudinal, rights-labeled, provenance-complete observations of external search reality and expose them through typed, LLM-readable tools.

The connected LLM interprets at read time.

The database itself never forms an opinion.

---

## What The Observatory Is Not

The Observatory is not:

- a strategy engine;
- a recommendation store;
- a customer database;
- a CRM;
- a dashboard project;
- a replacement for Kaizen, Neon Ronin, or SearchClarity;
- a place for customer first-party analytics;
- a place for agents to receive SQL or raw database credentials.

## Current Hard Rule

```text
The Observatory stores what was observed.
The connected LLM interprets what it means.
Accepted conclusions promote out to the owning consumer.
```

No schema, provider capture, MCP implementation, dashboard work, or strategy storage is active until the roadmap gates say so.
